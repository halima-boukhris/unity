# Mon Premier Projet Unity - Jeu de plateforme 2D

Ceci est mon premier projet réalisé sur Unity ! L'objectif est d'apprendre les bases du langage C#.

## Le Joueur
* **Animations** : Repos, course, saut et chute avec des animations dédiées (`Idle`, `Run`, `Jump`, `Fall`).
* **Combat** : Le joueur peut éliminer les ennemis en leur sautant dessus.
* **Système de dégâts** : Perte de vie au contact des ennemis avec un clignotement d'invulnérabilité.

## Interface (UI)
* **Barre de vie** : Elle diminue et change de couleur selon la santé restante.
* **Compteur** : Un texte affiche les points de vie en temps réel (ex: 5/5).
* **Menu Pause** : Possibilité de mettre le jeu en pause à tout moment.

## Technique
* **Version** : Unity 6 (6000.0.38f1).
* **Organisation** : Utilisation de ScriptableObjects pour gérer la vie et les événements de manière propre et indépendante.
